# Server-Performace-Stats

A script to analyze basic server performance stats on any linux system!

# How to use?

1. Clone the repository or copy the content of [Server-Stats](server-stats.sh)
```
git clone https://github.com/reneb1404/server-performace-stats
cd server-performance-stats
```
2. Make the script executable

```
chmod +x server-stats.sh
```

3. Execute the script

```
./server-stats.sh
```

## Alternative

1. Clone the repository or copy the content of [Server-Stats](server-stats.sh)

```
git clone https://github.com/reneb1404/server-performace-stats
cd server-performance-stats
```

2. Run the script in your terminal

```
bash server-stats.sh
```

3. The output will be shown in your CLI
